# SynqForge Custom Document Template - Starter Template

## Template Metadata
- **Name**: SynqForge Standard Story Format
- **Version**: 1.0
- **Description**: A starter template demonstrating the standard user story format with acceptance criteria

---

## User Story Title

**Format**: As a [persona], I want [goal], so that [benefit]

**Example**:
As a customer, I want to reset my password, so that I can regain access to my account when I forget it.

---

## Description

**Format**: Provide context about the user need and business value

**Example**:
Users frequently forget their passwords and need a secure way to reset them without contacting support. This feature reduces support burden and improves user experience by allowing self-service password recovery.

---

## Acceptance Criteria

**Format**: Given/When/Then format (Gherkin)

**Example**:
Given I am on the login page
When I click "Forgot Password"
Then I receive an email with a password reset link

Given I have received a password reset email
When I click the reset link
Then I am redirected to a secure password reset page

Given I am on the password reset page
When I enter a new password that meets security requirements
Then my password is updated and I can log in with the new password

---

## Priority

**Options**: critical | high | medium | low

**Example**: high

---

## Story Points

**Format**: Fibonacci sequence (1, 2, 3, 5, 8, 13)

**Example**: 5

## Template Instructions

1. **Required Sections**: Title, Description, and Acceptance Criteria are required
2. **Format Consistency**: Use consistent formatting throughout your template
3. **Examples**: Include example content to guide the AI
4. **Language**: This template uses US English. For UK English, use UK spelling (behaviour, colour, organise)
5. **Tone**: Adjust tone indicators (formal/casual) based on your needs

## How to Use This Template

1. Download this template

3. Add or remove sections as needed
4. Upload the customized template to SynqForge
5. Select it when generating stories
